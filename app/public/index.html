Source Control

Git + GitHub for version control. Every push triggers the CI/CD pipeline automatically via webhook.

Automated Testing

Unit tests run on every commit. Build fails fast if any test fails, preventing broken code from deploying.

GitHub Actions

Docker Build

Application is containerized using Docker. Ensures consistent environments from dev to production.

Docker
📦

Registry Push

Docker images are tagged and pushed to Docker Hub. Image versioning with git SHA for traceability.

Docker Hub

Cloud Deploy

Zero-downtime deployment to AWS EC2 via SSH. Rolling updates with automatic rollback on failure.
